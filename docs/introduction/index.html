Introduction

About this guide​

This guide provides an overview of IXLA's new software platform and integration guidelines for developers that need to work with our printers.

We will start with an overview of the system where you will learn about:

  • IXLA BOX our modular printers for mid-high volume production applications.
  • IWS (IXLA Web Service) the software platform that controls IXLA BOX and all new machines of the ID series.
  • IWS's WebApp, an example of software integration built on top of IWS's REST APIs

We will then setup and run workflows on IXLA BOX using the WebApp, see how to the same results can be achived with API calls and, finally, how to provide personalization data to the system and schedule jobs via database integration

note

Keep in mind that, although we will be using IXLA BOX as an example for integration, the same concepts apply to all supported systems by IWS

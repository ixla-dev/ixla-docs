Accessing REST APIs

By default IWS's HTTP service listens on port 5000 and binds to 0.0.0.0 (all IP addresses).

Open http://{BOX_IP_ADDRESS}:5000/swagger/index.html to see an exhaustive list of available endpoints in Swagger UI.

Generate SDK clients​

IWS's exposes API endpoint definitions in OpenAPI format. Use your favorite tool to generate SKD clients.
